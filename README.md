# 🖥️ Ops Dashboard

Dashboard de operaciones central con agentes IA y oficina pixel art.

## Features
- 📊 Gestión de proyectos con IA
- 💬 Sistema de prompts para agentes (Claude, Hermes, Grok)
- 💰 Tracking de costos
- 🎮 Oficina pixel art interactiva con agentes animados
- 🧠 Emociones dinámicas (bostezo, enojo, felicidad, sueño)

## Stack
- Single-file HTML/CSS/JS (~2100 líneas)
- Canvas 2D para pixel art
- Sin dependencias externas
- Sync con localStorage para estados de agentes

## Usage
Abrí `dashboard.html` en cualquier navegador. Todo está en un solo archivo.
